# MemoryAtlas

> Document-Driven Development with Atlas CLI

Quick flow (Intake -> Plan -> Finish):
1) `python atlas.py init` (first time)
2) `python atlas.py intake "..." --domain GEN`
3) `python atlas.py plan BRIEF-GEN-001`
4) `python atlas.py finish RUN-BRIEF-GEN-001-step-01 --git <hash|no-commit> --success true`
5) `python atlas.py doctor`

Top docs: FRONT.md, BOARD.md, CONVENTIONS.md, GOALS.md
