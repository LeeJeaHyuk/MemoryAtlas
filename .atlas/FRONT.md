# Atlas

This repo uses Atlas vNext.
Use: `python atlas.py init`

Quick flow:
1) `python atlas.py intake "..." --domain GEN`
2) `python atlas.py plan BRIEF-GEN-001`
3) `python atlas.py finish RUN-BRIEF-GEN-001-step-01 --git <hash|no-commit> --success true`
